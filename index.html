wtb

wtb is an acronym for what the box

wtb is an opensource JavaScript dimension parser

wtb parses dimensions and returns width, height, area, aspect ratio

wtb accepts dimensions in many formats

wtb is well coded and tested

wtb is fun and easy to use

wtb is made for Node.js and all web browsers

wtb is usable in the console on this page

Learn on Github or npm
